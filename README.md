# Expense Tracking System

A Java desktop application to efficiently manage and visualize personal expenses. Features a Swing-based GUI, secure authentication, full CRUD functionality, MySQL integration, and dynamic charts to analyze spending patterns.

---------------------------------------------------------------------------------------------------------------------------------
## Features

### User Authentication
- Secure login and registration system.
- Username and password validation.
- Individual accounts to track personal expenses separately.

### Expense Management
- Add new expenses with date, amount, and category.
- Edit or remove existing expenses.
- Supports custom categories.

### Category Management
- Add and delete expense categories.
- View all categories for easy selection while adding expenses.

### View & Search Spending
- View expenses date-wise or category-wise.
- Calculate total amount spent in a date range or category.
- Detailed table display of all relevant information.

### Visualization
- **Category Bar Chart:** Displays total expenses per category.
- **Date Line Chart:** Shows expense trends over time.
- Interactive charts for better insights.
